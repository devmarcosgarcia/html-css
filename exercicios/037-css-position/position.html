Positions CSS

Static

Static

Todos os elementos HTML por padrão seguem o posicionamento static.

Ou seja, eles seguem o fluxo que está disposto no documento HTML.

Relative

Com o relative podemos posicionar o elemento com as propriedades top, right, bottom, left.

Neste exemplo foi adicionado 10% de deslocamento do lado esquedo da div e 50px de deslocamento na parte superior.

O posicionamento relative se orienta por sua propria posição.
Mas, quando o elemento é deslocado, sua posicão original ainda fica ocupada por ele

Relative

Seu deslocamento pode sobrepor outros elementos.

Absolute

O posicionamento absolute é orientado pelo elemento pai. Neste caso o pai é o body.

Neste exemplo a div está flutuando sobre um parágrafo.

Para evitar que o elemento com posicionamento absolute tenha o body como pai.
Sempre é recomendado colocar o elemento absolute dentro de uma div relative.
